A personal repository to learn & practice solidity.

----
/
|-- basics/
|-- dapps/
|-- projects/
|-- security/
|-- testing/
|-- tooling/
|-- clear-deps.sh
----

basics:: Solidity language practice
dapps:: Dapps practice
projects:: Realization of mixed learned knowledge into a projet at a given time

testing:: solidity & web3 project testing practice
tooling:: tryout of tools related to solidity 
clear-deps.sh:: A cleaning script to get ride of node_modules through all the sub directories.
